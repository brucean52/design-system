## Usage Guidelines
The cascader component does not follow UX best practices. Use a traditional [dropdown](http://marklogic-ui-component-library.eng.marklogic.com/?path=/story/navigation-mldropdown--basic) whenever possible.

## Text Guidelines
For labels on all levels:
* If the hover text is the name of a command or function, use [Title Case](https://elementary.io/docs/human-interface-guidelines#capitalization).
* When a sentence is required, use [Sentence case](https://elementary.io/docs/human-interface-guidelines#capitalization).
* Use short, precise terms.
