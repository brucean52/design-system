## Usage Guidelines
A progress indicator should always be used to provide feedback to the user when an action can take awhile to complete.
### Progress Bar
* To provide feedback on operations that take a long time

### Mini Progress Bar
* When there are multiple progress bars
* If the action is quicker/smaller

### Mini Circular Progress Bar
* To communicate usage or quotas (e.g. how much storage the user has used up)
